# PRD 버전 히스토리

> 각 버전의 핵심 변경사항과 의사결정 기록. 상세 내용은 PRD v5.0(통합본)에 반영됨.

---

## v1.0 (2026.03.26)
**목적:** 초기 기획 — 교육용 데모 MVP

핵심 결정:
- React + Vite + Tailwind + Framer Motion 기술 스택 확정
- 에이전트 5명 구성 (Alex/Kai/Mia/Nova/Rex)
- 타이머 기반 JSON 시나리오 엔진 방식 채택
- 4개 Phase 구조 (인트로 → 킥오프 → 병렬작업 → 결과통합)
- Claude API 연동 (선택사항, mock fallback)

마일스톤: M1(뼈대) → M2(엔진) → M3(이펙트) → M4(API)

---

## v2.0 (2026.03.26)
**목적:** 시각적 몰입도 + 시나리오 현실성 강화

핵심 변경:
- 2D 아바타 & 아이소메트릭 오피스 배경 도입 결정
- 트러블슈팅 시나리오 추가 (API 차단 → 우회 해결)
- 결과물을 A4 비율 리포트 프리뷰로 업그레이드
- Phase 5 (후속 액션 — 슬랙/메일 배포) 추가

마일스톤: M5(비주얼) → M6(리치 시나리오) → M7(리포트+후속)

---

## v3.0 (2026.03.26)
**목적:** UI 폴리싱 + 전자동 파이프라인

핵심 변경:
- 사용자 선택 없이 전자동 흐름으로 전환
- ResultPanel → 중앙 팝업 모달 방식으로 변경
- 글로벌 B2B 다국어 캠페인 고난도 시나리오로 교체
- 에이전트 말풍선(statusText) 상시 표시
- ChatLog 발화자 이모지 + 키워드 컬러링

마일스톤: M8(UI 폴리싱) → M9(고난도 시나리오) → M10(팝업 결과물)

---

## v4.0 (2026.03.26)
**목적:** 데이터 대시보드 + 피날레 리텐션

핵심 변경:
- OfficeScene 축소 + DashboardPanel 영역 신설 (실시간 마케팅 지표)
- StatusPanel 높이 최적화 (5명 동시 표시)
- FinalSnsPopup에 "새 미션" / "반복 스케일링" 후속 버튼 추가

마일스톤: M11(레이아웃+대시보드) → M12(StatusPanel) → M13(후속 액션)

---

## v5.0 (2026.03.27) — 현재 통합본
**목적:** 전체 통합 + 대규모 개선

핵심 변경:
- v1~v4 + SCENARIO_DETAIL.md를 단일 PRD로 통합
- **2D 픽셀아트** 비주얼 시스템 확정 (스프라이트 시트, 타일맵, 걷기 애니메이션)
- **Human-in-the-loop (Decision Points 3개)** 추가 — 30초 타임아웃 + 자동 선택
- **루틴 업무 시스템** — Phase -1에서 에이전트가 이미 일하는 모습 표시
- **자체 판단 디테일** — 데이터 신뢰성, 로컬 시장 맥락, 예산 효율 자체 판단 장면
- **16:9 풀스크린 레이아웃** 확정 (프로젝터 시연 최적화)
- **ArtifactGallery** — DashboardPanel 대체, 에이전트별 실제 산출물 렌더링
- **언어 규칙** — 이름 영어, 대화/산출물 한국어
- **알림음** — Web Audio API, DecisionPopup 트리거 시

의사결정 기록:
- 3D 대신 2D 픽셀아트 선택 → 4/8 시연 마감 고려, 성능 안정성, 레트로 친근감
- DashboardPanel(지표 대시보드) → ArtifactGallery(산출물 갤러리)로 교체 → "에이전트가 뭘 만들었는지"가 핵심
- 전자동(v3) → HITL 혼합으로 복귀 → 교육 메시지 "사람+AI 협업"이 더 중요

마일스톤: M1(레이아웃+시나리오) → M2(HITL+루틴) → M3(픽셀아트) → M4(산출물) → M5(폴리싱)
