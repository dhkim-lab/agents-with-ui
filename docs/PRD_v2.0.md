# AI Marketing Team Office Demo — PRD v2.0 (개선판)

> 초기 v1.0 데모의 피드백을 수용하여 시각적 몰입도(2D 아바타, 사무실 씬), 시나리오의 현실성(트러블슈팅 및 작업 디테일 추가), 그리고 결과물의 완성도(리포트 미리보기 및 후속 액션)를 강화한 버전입니다.

| 버전 | 작성일 | 목적 |
|------|--------|------|
| v2.0 | 2026.03.26 | 교육/설득용 데모 몰입도 극대화 |

---

## 1. 개선 핵심 목표 (v2.0)

1. **시각적 몰입감 (Visual Upgrade)**: 단순한 도형 요소들을 실제 2D 게임 같은 아바타와 오피스 환경으로 업그레이드.
2. **에이전트 현실성 (Realistic Logs)**: "다 했습니다" 식의 단답형 메시지가 아닌, **문제 해결(Troubleshooting)과 구체적인 작업 내역**을 대화에 포함하여 "진짜 일하는 느낌" 부여.
3. **결과물 UX 강화 (Rich Result)**: 단순 텍스트가 아닌 실제 문서(PDF/노션) 형태의 미리보기를 제공하며, **외부 채널 공유(슬랙 메시지, 메일 전송 등)** 까지 이어지는 후속 세션 추가.

## 2. 세부 기능 명세

### 2.1 2D 아바타 & 사무실 배경 (UI/UX)
- **사무실 배경 고도화**: 단순 그리드가 아닌, 업무 공간(Desks), 식물(Plants), 화이트보드가 있는 회의실(Meeting Room) 등 아이소메트릭(Isometric) 또는 탑다운 2D 그래픽 씬으로 교체 (CSS/SVG 활용).
- **2D 아바타 애니메이션**: 
  - 단순 색상 원형이 아닌, 캐릭터 형태의 2D 아바타 적용 (SVG).
  - 상태별 애니메이션: 
    - `working`: 책상 앞에서 손을 움직이거나(타이핑), 머리가 까닥거리는 애니메이션
    - `idle`: 숨쉬기 
    - `waiting`: 시계를 보거나 머리에 물음표(❓)가 뜨는 애니메이션
- **역할 설명 툴팁**: 에이전트에 마우스를 올리면(Hover), 해당 에이전트의 구체적인 페르소나와 현재 맡은 태스크가 오버레이로 표시됨.

### 2.2 디테일한 작업 로그 & 트러블슈팅
시나리오 JSON 구성 시, 에이전트 간의 대화가 실제 실무자들의 커뮤니케이션처럼 보이도록 고도화.
- **이슈 발생 및 해결 루프**: 
  - *예시 (Kai)*: "타겟 소셜 데이터 크롤링 중 API Rate Limit 에러가 발생했습니다. 백오프(Back-off) 알고리즘을 적용해 우회 수집을 완료했습니다."
  - *예시 (Rex)*: "예산 최적화 시뮬레이션 중 A/B 테스트 지표에 이상치가 발견되어 클렌징 후 재분석했습니다. 최종 예상 CTR 3.2% 도출 완료."
- **상태 패널 고도화**: 진행률(%) 바 아래에 현재 처리 중인 로그 단위 과제(예: `[데이터 정제 중...]`, `[에러 복구 중...]`)가 실시간으로 스크롤링됨.

### 2.3 최종 결과물(Rich Preview) & 후속 세션(Session 2)
기존 Phase 4(결과물 통합) 이후에 **새로운 Phase 5 (후속 액션)** 를 추가.

- **리포트 미리보기 창**: 우측 하단이나 중앙 모달을 통해 실제 A4 비율의 "마케팅 플랜 리포트" 렌더링. (헤더, 차트 프레임, 목차 등 문서 템플릿 적용).
- **후속 액션 (Phase 5)**: 
  - 리포트가 완성되면 Alex(팀장)가 *"리포트가 완성되었습니다. 이해관계자에게 공유할까요?"* 라고 질문.
  - 사용자에게 2개의 버튼 제공:
    1. `[슬랙(Slack)으로 팀에 공유하기]`
    2. `[PDF로 추출하여 클라이언트 메일 발송]`
  - **버튼 클릭 시**: 에이전트 하나(예: Alex 또는 별도의 비서 에이전트)가 메신저 창을 띄우거나 메일을 작성하는 짧은 컷씬 애니메이션 실행 후 데모 완전 종료.
