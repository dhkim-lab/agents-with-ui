## 1. 개편 핵심 목표 (v3.0)

1. **상시 업무 가시성 (Agent Activity UI)**: 상태 아이콘 중복을 정리하고, 에이전트들이 각자 자리에서 '혼잣말' 혹은 '현재 작업(Status)'을 말풍선 형태로 띄워 현장감을 극대화.
2. **대화창(Chat) 가독성 리뉴얼**: 발화자 식별을 위한 이모지 추가 및 메시지 성격(에러, 성공, 일반)에 따른 색상 하이라이팅 적용.
3. **전자동(Auto-flow) 및 팝업형 결과물**: 
   - 결과물이 우측 패널에 고정되는 대신 중앙 팝업(Modal)으로 등장 (일정 시간 렌더링 후 자동 닫힘).
   - 사용자의 클릭을 기다리지 않고, 알아서 DB 적재 및 SNS 퍼블리싱까지 이어지는 **Phase 5 (DB/SNS 배포)** 자동 실행.
4. **고난도 시나리오 (Advanced Marketing Mission)**: 단순한 카피 작성을 넘어, "글로벌 B2B 다국어 캠페인 매체 세팅 및 퍼널 최적화" 등 무게감 있는 미션으로 시나리오 볼륨 업그레이드.

## 2. 세부 기능 및 화면 설계

### 2.1 아바타 및 UI 디테일 개선
- **오버랩 아이콘 수정**: AgentAvatar 우측 상단 뱃지(⚙️, ⏳, 🎉)가 여러 개 겹치지 않도록 상태 조건에 따라 1개만 노출시키고 위치 통일.
- **상태 말풍선(Speech Bubble)**: 에이전트 그래픽 위쪽으로 항상 `agent.statusText` 내용이 작은 말풍선으로 떠 있도록 수정. "데이터 크롤링 중...", "A/B 테스트 세팅 중" 등의 글귀가 각자 자리에서 표시됨.

### 2.2 지능형 채팅 로그 (Smart Chat Logs)
- **발화자 이모지**: "Alex" -> "🧢 Alex", "Kai" -> "🔍 Kai" 형태로 렌더링.
- **맥락 인식 컬러 옵션**:
  - `[ERROR]`, `⚠️`, `에러` 키워드 감지 시 텍스트 전체 혹은 테두리 **Red(빨강)**.
  - `[SUCCESS]`, `✅`, `해결`, `완료` 키워드 감지 시 텍스트 전체 **Green(초록)**.

### 2.3 모달형 결과물 & 전자동 후속 액션 (Phase 자동화)
- 기존의 우측에 공간을 차지하던 `ResultPanel`을 지우고, 화면 전체를 덮는 **팝업 모달(Popup Modal)** 형태로 리포트 노출.
- **자동 진행 로직**:
  - Phase 4에서 전략 플랜(PDF 팝업) 5초 노출 후 모달 자동 페이드아웃.
  - 바로 Phase 5 "시스템 연동 및 배포" (DB 적재, SNS 포스팅) 애니메이션 가동.
- **최종결과(SNS 팝업)**: 시나리오 마지막에 실제 '인스타그램' 유저 인터페이스를 본뜬 팝업 하나가 뜨며 캠페인 온에어 성공을 알리고 시연을 종료.
